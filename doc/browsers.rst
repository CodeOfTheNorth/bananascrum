##################
Supported browsers
##################

Every feature has to be tested on the following browsers

* Firefox 3.x
* Opera 9.x
* Internet Explorer 7, 8
* Safari 3.x, 4.x
* Chrome 1.x

Please note that we don't support Internet Explorer 6.
